### 1. 소개
맛있는 과일 쥬스(딸기쥬스, 바나나쥬스, 파인애플쥬스, 키위쥬스, 망고쥬스, 딸기바나나쥬스, 망고키위쥬스)를 주문하면, 해당 과일 쥬스를 만드는 데에 필요한 과일의 재고를 파악한 후에 재고가 있다면 맛있는 쥬스를 만들어줍니다. 과일의 재고가 없다면, 재고가 부족하다는 것을 알려주고 과일 재고를 수정할 수 있는 기능을 제공합니다.
